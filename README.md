#  ASP.NET Core Identity & JWT Authentication Boilerplate
This Boilerplate is designed to be independent of any database schema and to be as generic as possible.

### ASP.NET Core Identity Packages

Packages integrated into <a href="https://docs.microsoft.com/en-us/aspnet/identity/overview/getting-started/introduction-to-aspnet-identity" target="_blank">ASP.NET Core Identity</a> (supports .NET Standard).
